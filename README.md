# Blog, Todo Pages

## Requirements
```Django==4.1.5
django-autoslug==1.9.8
django-tinymce==3.5.0
python-slugify==8.0.0
```
## You can install packages with
```pip install -r requirements.txt```

## You can run the project with
```python manage.py runserver```
